Gateway Payment Button Examples

This page shows some of the many use-cases for the payment button. Most of these are, unfortunately, not implemented yet.

These are all possible with strictly the HTML button inject script usable on any website. If the user uses our React component, things like dynamically-generated buttons become possible.
