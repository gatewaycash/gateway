Gateway Payment Button Examples

This page shows some of the many use-cases for the payment button. Some of these are, unfortunately, not implemented yet.

These are all possible with strictly the HTML button inject script usable on any website. If the user uses our React component, things like dynamically-generated buttons become possible.

All of the buttons on this page will allow you to contribute to the Gateway project, except the one on the bottom, which won't generally work unless you're running a local payment server.
