Gateway Payment Button Examples

This page shows some of the many use-cases for the payment button. These are all possible with strictly the HTML button inject script usable on any website.

All of the buttons on this page will allow you to contribute to the Gateway project, except the one concerning Local payment server, which won't generally work unless you're running a local payment server for development.
